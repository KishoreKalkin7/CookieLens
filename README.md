# CookieLens 🔍🍪
A Chrome extension that uses AI to analyse cookie policies and show you a plain English risk score — built for Indian users under the DPDP Act 2023.

## How to install
1. Clone this repo
2. Add your Groq API key in background.js
3. Open chrome://extensions → Enable Developer Mode → Load Unpacked → select this folder

## Tech stack
Vanilla JavaScript · Chrome Manifest V3 · Groq API · LLaMA 3.1
